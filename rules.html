Simon Says Game

Rules To Play

The game starts when the player presses a key.

The game flashes the first color, e.g., "yellow."

The player clicks the "yellow" button

The game flashes the next sequence, e.g., "yellow, gree

The player clicks "yellow" and then "green."

The game adds another color to the sequence, e.g., "yellow, green, red," and so on.

The player must repeat the sequence in order to win the game.

If the player makes a mistake, the game ends and displays their score.

Good Luck!
